Manual de Soporte Técnico - MoviTrack Py
1. Introducción
Este documento detalla los procedimientos operativos estándar para la gestión de usuarios y el diagnóstico básico de dispositivos GPS.

2. Gestión de Clientes y Usuarios
Para asegurar la integridad de la base de datos, siga estos pasos:

Creación de usuario: Por cada cliente nuevo, se debe crear un usuario único en la plataforma. Nunca reutilice credenciales entre clientes distintos.

Búsqueda por Cédula: Para cualquier gestión administrativa o de soporte, busque al cliente siempre utilizando su número de cédula de identidad civil.

Búsqueda por Chapa: Para localizar un vehículo específico en el mapa, utilice el número de chapa. Este es el identificador primario para la unidad móvil.

3. Diagnóstico de Dispositivos GPS (Fallas Comunes)
Si un cliente reporta que el dispositivo no funciona, realice la siguiente verificación antes de derivar a soporte técnico avanzado:

Verificar conexión: Confirmar si el dispositivo reporta señal en la plataforma.

Revisión de alimentación: En caso de no reportar, verificar si el equipo tiene alimentación eléctrica (batería del vehículo/conexión).

Reporte de estado: Si el equipo sigue sin emitir datos tras verificar la alimentación, registrar el número de chapa y el ID del dispositivo para el ticket de revisión técnica en taller.

4. Política de Recuperación
Fin del servicio: Si el cliente decide no renovar o rescindir el contrato, se debe programar la recuperación física del equipo GPS dentro de los 5 días hábiles siguientes a la baja.

Morosidad: En caso de mora superior a 3 meses, se iniciará el protocolo de retiro forzoso del dispositivo instalado en el vehículo.
